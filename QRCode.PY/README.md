### Description

QRCode for python.


### Platform

Python 3 for Windows/Linux/MacOS.


### Install

```console
$ pip install qrcode
$ pip install pillow
```


### Feature

1. qr-write.py          ---- Create QR code picture.
2. qr-read.py           ---- Read QR information.


### Example:

1. QR write test:
```console
$ python qr-write.py

-----------Example 1-----------
Done 1.
-----------Example 1-----------
Done 2.
```

2. QR read test:
```console
$ python qr-read.py

```
