### Description

Timebase for python.


### Platform

Python 3 for Windows/Linux/MacOS.


### Install

```console
$ pip install ntplib
```

### Feature

1. timebase-ntp.py      ---- Get internet time from NTP server.


### Example:

1. Get internet time from ntp server test:
```console
$ python timebase-ntp.py

Sat Jul  9 20:46:09 2022
```
