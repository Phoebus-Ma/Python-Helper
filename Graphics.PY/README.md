### Description

Graphics programming for python.


### Platform

Python 3 for Windows/Linux/MacOS.


### Feature

1. graph-turtle.py          ---- Use turtle to draw lines.
2. graph-tkinter.py         ---- Use tkinter to gui programming.


### Example:

1. Turtle Test:
```console
$ python graph-turtle.py

# UI Window
```

2. Tkinter Test:
```console
$ python graph-tkinter.py

# UI Window
```
