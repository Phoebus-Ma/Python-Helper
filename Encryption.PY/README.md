### Description

Encryption test for python.



### Platform

Python 3 for Windows/Linux/MacOS.



### Feature

1. encrypt-md5.py   ---- MD5 encryption example.
2. encrypt-sha1.py  ---- SHA1 encryption example.



### Example:

1. MD5 Encryption Test:

```console
$ python encrypt-md5.py

```
